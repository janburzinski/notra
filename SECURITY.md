# Security Policy

## Reporting a Vulnerability

If you identify a security vulnerability, please contact us via email at security@usenotra.com or submit the details through our [GitHub Security Advisories](https://github.com/usenotra/notra/security/advisories) page. Our team will respond as promptly as possible.

We kindly request that you refrain from publicly disclosing any security issues until they have been resolved.

## Supported Versions

We provide security updates for the latest major release. Earlier versions may no longer be supported or eligible for updates.
